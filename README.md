# Enquete de Votação

Projeto em Java para simular uma enquete de votação.

## Funcionalidades

- Cadastro de candidatos
- Registro dos votos
- Resultado da votação
- Percentual de votos
- Exibição do vencedor

## Tecnologias

- Java
- Vetores
- Scanner

## Objetivo

Praticar vetores, laços de repetição e condicionais.
